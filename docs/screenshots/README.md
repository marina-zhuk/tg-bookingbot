# Скриншоты FitAdmin Bot

Эта папка содержит безопасные demo screenshots для portfolio case. Скриншоты нужны не как “доказательство готовой коробки”, а как визуальная витрина: клиент видит, какой Telegram flow можно собрать под его бизнес.

## Файлы

- `client-menu.png` — главное меню клиента.
- `subscription-flow.png` — выбор абонемента/услуги.
- `mock-payment.png` — demo payment flow без списания денег.
- `admin-panel.png` — панель администратора.
- `admin-subscriptions.png` — управление абонементами и ценами.
- `broadcasts.png` — сценарий рассылок.
- `crm.png` — CRM/список клиентов.
- `excel-report.png` — Excel/XLSX отчёт.

## Что важно перед публикацией

- Не показывать реальные телефоны и email клиентов.
- Не показывать реальные Telegram IDs администраторов.
- Не показывать bot tokens, payment keys, webhook URLs, private sheet/database links.
- Использовать demo-данные и safe placeholders.

## Минимальный demo flow для показа

1. Открыть landing page.
2. Перейти в Telegram demo bot.
3. Показать `/start` и регистрацию клиента.
4. Показать каталог абонементов.
5. Выбрать абонемент.
6. Показать demo successful payment.
7. Открыть admin panel.
8. Показать Excel report / broadcast / управление абонементами.
