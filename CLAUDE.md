# Botyara — Telegram Bot для спортивного клуба

## Идея проекта

Телеграм-бот для спортивного клуба, позволяющий участникам покупать и продлевать абонементы онлайн.

## Целевые пользователи

1. **Существующие клиенты** — уже есть в базе 1С, авторизуются и продлевают абонемент.
2. **Новые клиенты** — ещё не в базе 1С, регистрируются и покупают первый абонемент.

## Флоу пользователя

1. Открывает бота → авторизация (ФИО, номер телефона, email)
2. Выбирает подходящий абонемент (список типов/длительности/цен)
3. Оплата через кнопку (Telegram Payments или внешний эквайринг)
4. После оплаты — краткая инструкция: куда прийти, когда начать, что взять с собой
5. Рассылка: уведомления о скидках и акциях для всех пользователей бота

## Административная часть

- Список всех оплативших (с датой, типом абонемента, контактами)
- Интеграция с 1С: автосоздание профиля при оплате (если возможно через API 1С) или экспорт для ручного добавления менеджером
- Панель управления ценами и абонементами (менять без деплоя)

## Открытые вопросы

- **Договор**: подписывать онлайн до оплаты (КЭП/ПЭП) или при первом визите? — юридический вопрос
- **Интеграция 1С**: пока реализован CSV-экспорт для ручного добавления менеджером; REST API 1С не исследован

## Технический стек

- **Runtime**: Node.js >=22.0.0
- **Bot framework**: Telegraf
- **База данных**: SQLite (встроенный `node:sqlite`)
- **Платёжная система**: ЮKassa (подключена в коде, ещё не активирована)
